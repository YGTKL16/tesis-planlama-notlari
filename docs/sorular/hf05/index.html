    HF05 — Hücresel İmalat Soruları¶

    Nasıl kullan?

    Soruyu gör → cevaplamayı dene (kapalı kitap) → çözümü aç → hata kodla. Kapsam: DCA, ROC, Hollier, hücre maliyet analizi, entegre.

    S01 — Kolay — DCA (3×4)¶

    Soru

    Aşağıdaki makine-parça matrisi verilmiştir:

    P1 P2 P3 P4
    M1 1 0 1 0
    M2 0 1 1 0
    M3 1 0 0 1

    DCA yöntemini uygulayarak hücreleri belirleyin ve varsa istisna parçaları işaretleyin.

    Sezgi & Strateji

    Ne soruluyor: DCA adımlarıyla blok köşegen yapı bul. Hangi yöntem: DCA — satır/sütun toplamlarıyla sırala, 1'leri yoğunlaştır. Dikkat: P3 iki farklı makine ailesince kullanılıyor → istisna aday.

    Adım adım çözüm

    Verilen: 3×4 makine-parça matrisi. İstenen: DCA ile hücreler.

    Adım 1 — Satır ve sütun toplamları:

    P1 P2 P3 P4 Satır toplamı
    M1 1 0 1 0 2
    M2 0 1 1 0 2
    M3 1 0 0 1 2
    Sütun top. 2 1 2 1

    Satırlar azalan sırayla: tümü 2, sıra değişmez. Sütunlar artan sırayla: P2(1), P4(1), P1(2), P3(2).

    Adım 2 — Ön sıralama:

    P2 P4 P1 P3
    M1 0 0 1 1
    M2 1 0 0 1
    M3 0 1 1 0

    Adım 3 — İlk satır (M1) 1'li sütunları sola taşı → P1, P3:

    P1 P3 P2 P4
    M1 1 1 0 0
    M2 0 1 1 0
    M3 1 0 0 1

    Adım 4 — İlk sütun (P1) 1'li satırları yukarı taşı → M1, M3:

    P1 P3 P2 P4
    M1 1 1 0 0
    M3 1 0 0 1
    M2 0 1 1 0

    Adım 5 — İkinci satır (M3) 1'li sütunları (P4) kalan sütunlar içinde sola taşı:

    P1 P3 P4 P2
    M1 1 1 0 0
    M3 1 0 1 0
    M2 0 1 0 1

    Blok analizi:

    • Sol-üst blok: M1,M3 × P1 (P3'ü de M1 kullanıyor, P4'ü de M3 kullanıyor)
    • Sağ-alt blok: M2 × P2 (P3'ü de M2 kullanıyor)

    Karar: - Hücre 1: M1, M3 — parçalar P1, P4 - Hücre 2: M2 — parça P2 - İstisna: P3 — hem Hücre 1'deki M1'i hem Hücre 2'deki M2'yi kullanıyor → hücreler arası taşıma gerektirir.

    S02 — Orta — ROC (4×5)¶

    Soru

    Aşağıdaki 4 makine × 5 parça matrisi verilmiştir:

    P1 P2 P3 P4 P5
    M1 1 1 0 0 0
    M2 0 1 1 0 0
    M3 0 0 0 1 1
    M4 0 0 0 1 0

    ROC yöntemini uygulayarak (en az 2 iterasyon) makine-parça hücrelerini belirleyin.

    Sezgi & Strateji

    Ne soruluyor: ROC ile blok köşegen yapı oluştur. Hangi yöntem: Binary ağırlık → azalan sıra → satır + sütun → yeniden sırala. Dikkat: Ağırlık yönü: en sol/üst sütun en yüksek 2^(n-1) ağırlığı taşır; azalan sıralama yapılır.

    Adım adım çözüm

    Verilen: 4×5 matris. İstenen: ROC ile hücreler.

    Iterasyon 1 — Satır ağırlıkları (5 sütun → ağırlıklar 16, 8, 4, 2, 1):

    Satır P1(16) P2(8) P3(4) P4(2) P5(1) \(R_i\)
    M1 1 1 0 0 0 24
    M2 0 1 1 0 0 12
    M3 0 0 0 1 1 3
    M4 0 0 0 1 0 2

    Azalan sıra: M1(24) → M2(12) → M3(3) → M4(2)

    Yeniden sıralanmış matris:

    P1 P2 P3 P4 P5
    M1 1 1 0 0 0
    M2 0 1 1 0 0
    M3 0 0 0 1 1
    M4 0 0 0 1 0

    Sütun ağırlıkları (4 satır → ağırlıklar 8, 4, 2, 1):

    Sütun M1(8) M2(4) M3(2) M4(1) \(C_j\)
    P1 1 0 0 0 8
    P2 1 1 0 0 12
    P3 0 1 0 0 4
    P4 0 0 1 1 3
    P5 0 0 1 0 2

    Azalan sıra: P2(12) → P1(8) → P3(4) → P4(3) → P5(2)

    Iterasyon 1 sonucu:

    P2 P1 P3 P4 P5
    M1 1 1 0 0 0
    M2 1 0 1 0 0
    M3 0 0 0 1 1
    M4 0 0 0 1 0

    Iterasyon 2 — Satır ağırlıkları (ağırlıklar 16, 8, 4, 2, 1): M1: 16+8=24, M2: 16+4=20, M3: 2+1=3, M4: 2 Sıra: M1(24) → M2(20) → M3(3) → M4(2) → değişmedi.

    Sütun ağırlıkları (ağırlıklar 8, 4, 2, 1): P2: 8+4=12, P1: 8, P3: 4, P4: 2+1=3, P5: 2 Sıra aynı → yakınsadı.

    Blok analizi:

         P2  P1 | P3 | P4  P5
M1 [  1   1 |  0 |  0   0 ]  ← Hücre 1
M2 [  1   0 |  1 |  0   0 ]  ← Hücre 1
────────────────────────────
M3 [  0   0 |  0 |  1   1 ]  ← Hücre 2
M4 [  0   0 |  0 |  1   0 ]  ← Hücre 2

    Karar: - Hücre 1: M1, M2 — parçalar P1, P2 (ve P3 — M2 kullanıyor ama H2'de yok → Hücre 1 dahili) - Hücre 2: M3, M4 — parçalar P4, P5 - P3 yalnızca M2 (Hücre 1) tarafından kullanılıyor → istisna değil, Hücre 1'e aittir.

    S03 — Orta — Hollier Yöntemi (5 makine)¶

    Soru

    Beş makineli bir hücrede from-to akış değerleri aşağıdaki gibidir:

    Kaynak → Hedef Akış
    M1 → M2 30
    M1 → M3 20
    M2 → M4 25
    M3 → M4 15
    M3 → M5 10
    M4 → M5 35

    (a) Her makinenin G/A oranını hesaplayın. (b) Hollier sıralamasını bulun. (c) Bu sırada ileri sıralı, atlamalı ve geri akış yüzdelerini hesaplayın.

    Sezgi & Strateji

    Ne soruluyor: Hollier G/A oranları → sıralama → akış sınıflandırması. Hangi yöntem: \(G_i = \sum f_{i\to}\), \(A_i = \sum f_{\to i}\), \(r_i = G_i/A_i\) → azalan sıra. Dikkat: \(A_i=0,\ G_i>0\) → oran \(\infty\) → makine başa gider. Sıralama sonrası pozisyona bakarak ileri/atlamalı/geri belirle.

    Adım adım çözüm

    Verilen: 5 makine, 6 yönlü akış. İstenen: oranlar, sıra, akış yüzdeleri.

    Adım 1 — G ve A toplamları:

    Makine Gidiş \(G_i\) Geliş \(A_i\) Oran \(r_i\)
    M1 30+20=50 0 ∞
    M2 25 30 0,833
    M3 15+10=25 20 1,250
    M4 35 25+15=40 0,875
    M5 0 10+35=45 0

    Adım 2 — Azalan sıra → Hollier dizisi:

    M1(∞) → M3(1,25) → M4(0,875) → M2(0,833) → M5(0)

    Pozisyonlar: M1=1, M3=2, M4=3, M2=4, M5=5

    Adım 3 — Her akışı sınıflandır:

    Akış Miktar Kaynak pos. Hedef pos. Sınıf
    M1→M2 30 1 4 Atlamalı (M3,M4 atlıyor)
    M1→M3 20 1 2 İleri sıralı
    M2→M4 25 4 3 Geri
    M3→M4 15 2 3 İleri sıralı
    M3→M5 10 2 5 Atlamalı (M4,M2 atlıyor)
    M4→M5 35 3 5 Atlamalı (M2 atlıyor)

    Adım 4 — Toplamlar:

    Sınıf Toplam akış Yüzde
    İleri sıralı 20+15=35 35/135 = %25,9
    Atlamalı 30+10+35=75 75/135 = %55,6
    Geri 25 25/135 = %18,5
    Toplam 135 %100

    Karar: Dizide M2→M4 geri akışı (%18,5) oluşuyor. Geri akışı azaltmak için M2 ile M4'ün pozisyonu yer değiştirilebilir; ancak bu diğer akışları bozabilir — önce toplam maliyeti karşılaştır.

    S04 — Orta — Hücre Maliyet Analizi ve İstisna¶

    Soru

    Altı parça ve beş makineli bir fabrikada iki hücre oluşturulmuştur:

    M1 M2 M3 M4 M5
    P1 1 1 0 0 0
    P2 0 1 1 0 0
    P3 1 0 0 1 0
    P4 0 0 1 0 0
    P5 0 0 0 1 1
    P6 0 0 0 0 1

    Hücre 1: M1, M2, M3 | Hücre 2: M4, M5 Üretim miktarları: P1=100, P2=80, P3=50, P4=60, P5=120, P6=90

    (a) Hangi parça istisnadır ve neden? (b) Hücre içi ve hücre dışı akış yüzdelerini (hareket sayısına göre) bulun. (c) Maliyet açısından yorumlayın.

    Sezgi & Strateji

    Ne soruluyor: İstisna parça tespiti + hücre içi/dışı akış oranı. Hangi yöntem: Her parçanın hangi hücre makinelerini kullandığına bak. İki hücreye değen = istisna. Dikkat: Akış sayısı: her üretim biriminin hücreler arası hareketi "1 hücre dışı hareket" sayılır.

    Adım adım çözüm

    Verilen: 6×5 matris, 2 hücre, üretim miktarları. İstenen: istisna + akış yüzdeleri.

    Adım 1 — Her parçanın hücre üyeliği:

    Parça Kullandığı makineler Hücre üyeliği
    P1 M1(H1), M2(H1) Hücre 1 ✓
    P2 M2(H1), M3(H1) Hücre 1 ✓
    P3 M1(H1), M4(H2) İSTİSNA
    P4 M3(H1) Hücre 1 ✓
    P5 M4(H2), M5(H2) Hücre 2 ✓
    P6 M5(H2) Hücre 2 ✓

    P3 istisnadır — hem Hücre 1'deki M1'i hem Hücre 2'deki M4'ü kullanıyor; üretim sırasında hücreler arası taşıma gerektiriyor.

    Adım 2 — Hücre içi ve dışı hareket:

    Parça Miktar Durum Hücre içi Hücre dışı
    P1 100 H1 içi 100 0
    P2 80 H1 içi 80 0
    P3 50 İstisna 0 50
    P4 60 H1 içi 60 0
    P5 120 H2 içi 120 0
    P6 90 H2 içi 90 0
    Toplam 500 450 50
    \[\text{Hücre içi oran} = \frac{450}{500} = \%90$$ $$\text{Hücre dışı oran} = \frac{50}{500} = \%10\]

    Karar: Hücre içi oran %90 — iyi bir hücreleme. İstisna P3'ün çözüm seçenekleri: (1) P3'ü fason/dışarı ver, (2) M4'ün bir kopyasını Hücre 1'e ekle, (3) P3'ü yeniden tasarla (M4 kullanmasın).

    S05 — Zor — Entegre (ROC + İstisna + Hollier)¶

    Soru

    Aşağıdaki 4 makine × 6 parça matrisi verilmiştir. Üretim rotaları (akış miktarıyla) da aşağıda tanımlanmıştır.

    Makine-parça matrisi:

    P1 P2 P3 P4 P5 P6
    M1 1 1 0 0 0 0
    M2 0 1 1 0 0 0
    M3 0 0 0 1 1 0
    M4 0 0 1 1 0 1

    From-to akışları (hücre içi, P3 hariç):

    Rota Miktar
    M1→M2 (P1+P2) 130
    M4→M3 (P4+P6) 90
    M3→M4 (P5) 40

    (a) ROC yöntemi ile hücreleri bulun (2 iterasyon). (b) İstisna parçaları belirleyin. (c) Her hücre için Hollier sıralamasını yapın ve geri akış yüzdesini hesaplayın.

    Sezgi & Strateji

    Ne soruluyor: ROC → hücre belirleme → Hollier. Hangi yöntem: Önce ROC, sonra her hücre kendi from-to'suyla Hollier. Dikkat: 2-makineli hücrede Hollier çok hızlı biter. 2-makineli hücrede geri akış = çift yönlü akışın küçük olanı.

    Adım adım çözüm

    Verilen: 4×6 matris + from-to akışları. İstenen: ROC hücreler + istisna + Hollier.

    (a) ROC — Iterasyon 1

    Satır ağırlıkları (6 sütun → 32,16,8,4,2,1):

    P1(32) P2(16) P3(8) P4(4) P5(2) P6(1) \(R_i\)
    M1 1 1 0 0 0 0 48
    M2 0 1 1 0 0 0 24
    M3 0 0 0 1 1 0 6
    M4 0 0 1 1 0 1 13

    Sıra: M1(48) → M2(24) → M4(13) → M3(6)

    Sütun ağırlıkları (4 satır → 8,4,2,1):

    M1(8) M2(4) M4(2) M3(1) \(C_j\)
    P1 1 0 0 0 8
    P2 1 1 0 0 12
    P3 0 1 1 0 6
    P4 0 0 1 1 3
    P5 0 0 0 1 1
    P6 0 0 1 0 2

    Sıra: P2(12) → P1(8) → P3(6) → P4(3) → P6(2) → P5(1)

    Iterasyon 1 sonucu:

    P2 P1 P3 P4 P6 P5
    M1 1 1 0 0 0 0
    M2 1 0 1 0 0 0
    M4 0 0 1 1 1 0
    M3 0 0 0 1 0 1

    Iterasyon 2 — Satır ağırlıkları: M1=32+16=48, M2=32+8=40, M4=8+4+2=14, M3=4+1=5 → sıra değişmez. Sütun ağırlıkları: P2=12, P1=8, P3=6, P4=3, P6=2, P5=1 → sıra değişmez → yakınsadı.

    Blok yapısı:

         P2  P1 | P3 | P4  P6  P5
M1 [  1   1 |  0 |  0   0   0 ]  ← Hücre 1
M2 [  1   0 |  1 |  0   0   0 ]  ← Hücre 1
─────────────────────────────────
M4 [  0   0 |  1 |  1   1   0 ]  ← Hücre 2
M3 [  0   0 |  0 |  1   0   1 ]  ← Hücre 2
    • Hücre 1: M1, M2 — parçalar P1, P2
    • Hücre 2: M3, M4 — parçalar P4, P5, P6

    (b) İstisna: P3

    P3 hem M2 (Hücre 1) hem M4 (Hücre 2) tarafından kullanılıyor → hücreler arası geçiş gerektirir.

    (c) Hollier sıralaması

    Hücre 1 (M1, M2): from-to: M1→M2=130

    Makine G A r
    M1 130 0 ∞
    M2 0 130 0

    Sıra: M1 → M2 (tek yönlü akış, geri akış yoktur) İleri sıralı: 130/130 = %100, Geri: %0

    Hücre 2 (M3, M4): from-to: M4→M3=90, M3→M4=40

    Makine G A r
    M3 40 90 0,44
    M4 90 40 2,25

    Azalan sıra: M4(2,25) → M3(0,44)

    Akış Miktar Pozisyon Sınıf
    M4→M3 90 1→2 İleri sıralı
    M3→M4 40 2→1 Geri

    Toplam = 130 İleri sıralı: 90/130 = %69,2 | Geri: 40/130 = %30,8

    Karar: Hücre 2'de P5 (M3→M4) geri akış oluşturuyor. Alternatif sıra M3→M4 denenirse P4 ve P6 geri akışa girer — mevcut sıra toplam geri akışı minimize ediyor.
